1. ภาพรวมของระบบ
================================================================================

JusBid เป็นแพลตฟอร์มประมูลสินค้าออนไลน์ที่รองรับการใช้งาน 3 บทบาท ได้แก่
ผู้ประมูล (Bidder), ผู้ขาย (Seller) และผู้ดูแลระบบ (Admin)

ระบบรองรับฟังก์ชันหลักดังนี้:
  - การลงทะเบียน/เข้าสู่ระบบ ด้วย JWT Authentication
  - การสร้าง/จัดการ/ประมูลสินค้าแบบเรียลไทม์
  - ระบบแชทระหว่างผู้ซื้อและผู้ขาย (รองรับข้อความและรูปภาพ)
  - ระบบให้คะแนนและรายงานผู้ขาย
  - แผงควบคุมสำหรับผู้ดูแลระบบ
  - การแจ้งเตือนผ่านอีเมล

เทคโนโลยีที่ใช้:
  - Backend:  Node.js + Express.js
  - Frontend: React.js + Vite
  - Database: PostgreSQL
  - Auth:     JSON Web Token (JWT)


================================================================================
2. บทบาทผู้ใช้งาน (User Roles)
================================================================================

ระบบแบ่งผู้ใช้ออกเป็น 3 บทบาท:

  (1) ผู้ประมูล (Bidder / Buyer)
      - ดูและค้นหาสินค้าที่กำลังประมูล
      - เข้าร่วมประมูลสินค้า
      - ดูรายการที่ชนะประมูล
      - ให้คะแนน/รายงานผู้ขาย
      - แชทกับผู้ขาย

  (2) ผู้ขาย (Seller)
      - สร้างรายการประมูลสินค้า
      - แก้ไขรายการก่อนเริ่มประมูล
      - ดูรายการประมูลของตนเอง (กำลังดำเนินอยู่/จบแล้ว)
      - แชทกับผู้ชนะประมูล
      - ดูสถิติและคะแนนของตนเอง

  (3) ผู้ดูแลระบบ (Admin)
      - ดูสถิติภาพรวมของแพลตฟอร์ม
      - จัดการบัญชีผู้ใช้ (ระงับ/เปิดใช้งาน)
      - จัดการรายงานปัญหา
      - ดูบันทึกการประมูลทั้งหมด
      - ส่งข้อความถึงผู้ใช้

3. การสมัครสมาชิก (Register)
================================================================================

วิธีการ:
  1) เข้าหน้าเว็บ แล้วกดปุ่ม "Register" ที่มุมขวาบน
  2) กรอกข้อมูลที่จำเป็น:
     - Display Name  : ชื่อที่แสดง (ตั้งได้ตามต้องการ)
     - Email          : อีเมล (ต้องไม่ซ้ำกับผู้ใช้อื่น)
     - Username       : ชื่อผู้ใช้ (ต้องไม่ซ้ำกับผู้ใช้อื่น)
     - Password       : รหัสผ่าน
     - Role           : เลือกบทบาท "Bidder" หรือ "Seller"

  3) กดปุ่ม "Register" เพื่อสร้างบัญชี

เงื่อนไขรหัสผ่าน:
  - ต้องมีความยาวอย่างน้อย 8 ตัวอักษร
  - ต้องมีตัวอักษรพิมพ์เล็กอย่างน้อย 1 ตัว
  - ต้องมีตัวอักษรพิมพ์ใหญ่อย่างน้อย 1 ตัว
  - ต้องมีตัวเลขอย่างน้อย 1 ตัว

เงื่อนไขอื่นๆ:
  - อีเมลต้องอยู่ในรูปแบบที่ถูกต้อง
  - Username และ Email ต้องไม่ซ้ำกับบัญชีที่มีอยู่ในระบบ
  - ต้องกรอกข้อมูลทุกช่องให้ครบ

เมื่อสมัครสำเร็จ:
  - ระบบจะแสดงข้อความ "Registration successful"
  - สามารถเข้าสู่ระบบได้ทันที


================================================================================
4. การเข้าสู่ระบบ (Login)
================================================================================

วิธีการ:
  1) กดปุ่ม "Login" ที่แถบนำทาง (Navbar) มุมขวาบน
  2) กรอกข้อมูล:
     - Username : ชื่อผู้ใช้ที่ลงทะเบียนไว้
     - Password : รหัสผ่าน
  3) กดปุ่ม "Login"

หลังเข้าสู่ระบบสำเร็จ:
  - ระบบจะนำทางไปยังหน้าหลักตามบทบาท:
    * Bidder  -> หน้า Home Bidder
    * Seller  -> หน้า Home Seller
    * Admin   -> หน้า Admin Dashboard
  - Token จะถูกเก็บใน Session Storage (มีอายุ 1 ชั่วโมง)

กรณีเข้าสู่ระบบไม่สำเร็จ:
  - Username/Password ไม่ถูกต้อง -> แสดงข้อผิดพลาด
  - บัญชีถูกระงับ (Suspended) -> แสดงข้อความแจ้งเตือนและไม่สามารถเข้าใช้งานได้


================================================================================
5. การออกจากระบบ (Logout)
================================================================================

วิธีการ:
  1) คลิกที่ไอคอนผู้ใช้ (มุมขวาบนของ Navbar)
  2) เลือก "Logout" จากเมนูดร็อปดาวน์
  3) ระบบจะลบ Token, userId, role ออกจาก Session Storage
  4) นำทางกลับไปหน้า Landing Page


================================================================================
6. หน้าแรก (Landing Page) - ผู้เยี่ยมชม
================================================================================

เมื่อเข้าเว็บไซต์โดยไม่ได้ล็อกอิน จะเห็นหน้า Landing Page ซึ่งแสดง:
  - แบนเนอร์แนะนำแพลตฟอร์ม
  - รายการสินค้าที่กำลังประมูลอยู่ในขณะนี้
  - ปุ่ม Login และ Register สำหรับเข้าสู่ระบบ/สมัครสมาชิก

ผู้เยี่ยมชม (Guest) สามารถ:
  - ดูรายการประมูลที่กำลังดำเนินอยู่
  - ดูรูปภาพ ชื่อ ราคาเริ่มต้นของสินค้า
  - ไม่สามารถประมูลหรือแชทได้ (ต้องสมัครสมาชิกก่อน)

7. ฟังก์ชันสำหรับผู้ประมูล (Bidder)
================================================================================

--------------------------------------------------------------------------------
7.1 หน้าหลักผู้ประมูล (Home Bidder)
--------------------------------------------------------------------------------
  - แสดงรายการสินค้าประมูลทั้งหมดที่กำลังดำเนินอยู่
  - แต่ละรายการแสดง: รูปสินค้า, ชื่อ, ราคาปัจจุบัน, เวลาที่เหลือ
  - มีตัวนับเวลาถอยหลัง (Countdown Timer) แบบ HH:MM:SS
  - ข้อมูลราคาอัปเดตอัตโนมัติทุก 2 วินาทีขณะประมูลดำเนินอยู่

--------------------------------------------------------------------------------
7.2 การค้นหาและกรองสินค้า
--------------------------------------------------------------------------------
  ค้นหา:
    - ใช้ช่องค้นหาที่แถบนำทาง (Navbar) 
    - พิมพ์ชื่อสินค้าเพื่อค้นหาแบบเรียลไทม์

  กรองตามหมวดหมู่ (15 หมวดหมู่):
    1)  Electronics        (อิเล็กทรอนิกส์)
    2)  Fashion            (แฟชั่น)
    3)  Art                (ศิลปะ)
    4)  Vehicles           (ยานพาหนะ)
    5)  Jewelry            (เครื่องประดับ)
    6)  Watches            (นาฬิกา)
    7)  Sports             (กีฬา)
    8)  Furniture          (เฟอร์นิเจอร์)
    9)  Books              (หนังสือ)
    10) Music              (ดนตรี)
    11) Toys               (ของเล่น)
    12) Beauty             (ความงาม)
    13) Collectibles       (ของสะสม)
    14) Home & Garden      (บ้านและสวน)
    15) Other              (อื่นๆ)

  วิธีใช้:
    - คลิกที่ปุ่มหมวดหมู่ที่ต้องการ
    - ระบบจะแสดงเฉพาะสินค้าในหมวดหมู่นั้น
    - กดอีกครั้งเพื่อยกเลิกการกรอง

--------------------------------------------------------------------------------
7.3 การดูรายละเอียดสินค้า
--------------------------------------------------------------------------------
  คลิกที่การ์ดสินค้า จะเปิด Popup แสดง:
    - รูปภาพสินค้าขนาดใหญ่
    - ชื่อสินค้า
    - รายละเอียด/คำอธิบาย
    - ชื่อผู้ขาย (คลิกเพื่อดูโปรไฟล์ผู้ขาย)
    - ราคาประมูลปัจจุบัน
    - ขั้นต่ำในการเพิ่มราคา (Bid Increment)
    - เวลาที่เหลือ
    - ช่องกรอกราคาที่ต้องการประมูล
    - ปุ่ม "Place Bid"

--------------------------------------------------------------------------------
7.4 การประมูลสินค้า (Bidding)
--------------------------------------------------------------------------------
  วิธีการ:
    1) คลิกปุ่ม "Bid" หรือ "Place Bid" ที่การ์ดสินค้า
    2) จะเปิด Popup แสดงข้อมูลสินค้า
    3) กรอกจำนวนเงินที่ต้องการประมูลในช่อง "Your Bid"
    4) กดปุ่ม "Place Bid" เพื่อยืนยัน

  เงื่อนไขการประมูล:
    - สามารถประมูลได้เฉพาะช่วงเวลาที่ประมูลกำลังดำเนินอยู่เท่านั้น
      (start_time <= เวลาปัจจุบัน < end_time)
    - ราคาที่เสนอต้อง >= ราคาปัจจุบัน + ขั้นต่ำ (bid increment)
      เช่น ราคาปัจจุบัน 1000 บาท, ขั้นต่ำ 100 บาท
      -> ต้องเสนออย่างน้อย 1100 บาท
    - ราคาที่เสนอต้องเป็นจำนวนเต็ม (ไม่มีทศนิยม)
    - ไม่สามารถประมูลสินค้าของตัวเอง (สำหรับ Seller)

  เมื่อประมูลสำเร็จ:
    - ราคาจะอัปเดตทันที
    - ผู้ประมูลรายอื่นจะเห็นราคาใหม่ภายใน 2 วินาที
    - แสดงข้อความ "Bid placed successfully"

  การ์ดสินค้าจะแสดงสถานะตาม 3 ระยะ:
    - ก่อนเริ่ม:  แสดงเวลาเริ่มประมูล
    - ระหว่างประมูล: แสดงเวลาถอยหลังและปุ่ม Bid
    - หลังจบ: แสดงชื่อผู้ชนะและราคาสุดท้าย

--------------------------------------------------------------------------------
7.5 การดูรายการที่ชนะประมูล
--------------------------------------------------------------------------------
  วิธีการ:
    1) เข้าหน้า Profile โดยคลิกไอคอนผู้ใช้ -> "Profile"
    2) จะเห็นส่วน "Won Auctions" แสดงรายการทั้งหมดที่เคยชนะ
    3) แต่ละรายการแสดง:
       - รูปสินค้า
       - ชื่อสินค้า
       - ราคาที่ชนะ
       - รายละเอียดการประมูล

  ขั้นตอนหลังชนะประมูล:
    - ติดต่อผู้ขายผ่านระบบแชท
    - ตกลงวิธีชำระเงิน
    - ตกลงวิธีส่งสินค้า
    - ตรวจสอบสินค้า

--------------------------------------------------------------------------------
7.6 การดูโปรไฟล์ผู้ขาย (Public Profile)
--------------------------------------------------------------------------------
  วิธีการ:
    1) คลิกชื่อผู้ขายในรายละเอียดสินค้า
    2) จะนำไปยังหน้า Public Profile ของผู้ขาย
    3) แสดงข้อมูล:
       - รูปโปรไฟล์
       - Display Name
       - Username
       - จำนวนรายการประมูลที่สร้าง
       - คะแนน Like/Dislike จากผู้ซื้อ
       - วันที่สมัครสมาชิก
       - รายการสินค้าทั้งหมดของผู้ขาย

--------------------------------------------------------------------------------
7.7 การให้คะแนนผู้ขาย (Like / Dislike)
--------------------------------------------------------------------------------
  วิธีการ:
    1) เข้าไปที่หน้า Public Profile ของผู้ขาย
    2) กดปุ่ม "Like" (👍) หรือ "Dislike" (👎)
    3) สามารถเปลี่ยนการให้คะแนนได้ภายหลัง
       (เช่น เคยกด Like แล้ว สามารถเปลี่ยนเป็น Dislike ได้)

  หมายเหตุ:
    - ผู้ใช้ 1 คนสามารถให้คะแนนผู้ขาย 1 คนได้เพียง 1 ครั้ง
      (แต่เปลี่ยนได้)
    - คะแนนจะแสดงเป็นจำนวน Like สีเขียว และ Dislike สีแดง

--------------------------------------------------------------------------------
7.8 การรายงานผู้ขาย (Report)
--------------------------------------------------------------------------------
  วิธีการ:
    1) เข้าไปที่หน้า Public Profile ของผู้ขาย
    2) กดปุ่ม "Report"
    3) เลือกประเภทปัญหา:
       - Auction Fraud          (การฉ้อโกงในการประมูล)
       - Non-delivery           (ไม่ส่งสินค้า)
       - Item Mismatch          (สินค้าไม่ตรงตามที่สั่ง)
       - Inappropriate Behavior (พฤติกรรมไม่เหมาะสม)
       - Other                  (อื่นๆ)
    4) กรอกรายละเอียดเพิ่มเติม
    5) กดปุ่ม Submit เพื่อส่งรายงาน

  เมื่อส่งรายงาน:
    - รายงานจะถูกส่งไปยังผู้ดูแลระบบ (Admin)
    - สถานะเริ่มต้นเป็น "Pending"
    - Admin จะตรวจสอบและดำเนินการ

--------------------------------------------------------------------------------
7.9 การแชท (Chat) - สำหรับผู้ประมูล
--------------------------------------------------------------------------------
  เมื่อชนะประมูลหรือต้องการติดต่อผู้ขาย:
    1) กดปุ่ม "Chat" หรือ "💬" ที่การ์ดสินค้า/หน้ารายละเอียด
    2) จะเปิดหน้าแชทสำหรับรายการประมูลนั้นๆ
    3) สามารถ:
       - ส่งข้อความตัวอักษร
       - ส่งรูปภาพ (กดไอคอนแนบไฟล์)
       - ดูประวัติการสนทนา
    4) กด Enter เพื่อส่ง, Shift+Enter เพื่อขึ้นบรรทัดใหม่

  (ดูรายละเอียดเพิ่มเติมในหัวข้อ 10: ระบบแชท)

--------------------------------------------------------------------------------
7.10 การจัดการโปรไฟล์ (Bidder Profile)
--------------------------------------------------------------------------------
  เข้าถึง: คลิกไอคอนผู้ใช้ -> "Profile"

  หน้าโปรไฟล์แสดง:
    - รูปโปรไฟล์
    - Display Name
    - Email
    - วันที่สมัครสมาชิก
    - รายการที่ชนะประมูล (Won Auctions)

  แก้ไขโปรไฟล์:
    1) กดปุ่ม "Edit Profile"
    2) จะเปิด Modal แก้ไข ให้สามารถเปลี่ยน:
       - Display Name (ชื่อที่แสดง)
       - Email
       - Password (รหัสผ่าน - มีเงื่อนไข 8 ตัวอักษรขึ้นไป)
       - Profile Image (อัปโหลดรูปโปรไฟล์ใหม่ พร้อม Preview)
    3) กดปุ่ม "Save" เพื่อบันทึกการเปลี่ยนแปลง

  หมายเหตุ:
    - สามารถแก้ไขบางช่องโดยไม่ต้องแก้ทุกช่อง
    - รูปภาพจะแสดง Preview ก่อนบันทึก
    - หากเปลี่ยน Password ต้องเป็นไปตามเงื่อนไขรหัสผ่าน

8. ฟังก์ชันสำหรับผู้ขาย (Seller)
================================================================================

--------------------------------------------------------------------------------
8.1 หน้าหลักผู้ขาย (Home Seller)
--------------------------------------------------------------------------------
  มีระบบสลับมุมมอง 2 แบบ:
    (ก) "All Auctions" - แสดงรายการประมูลทั้งหมดในแพลตฟอร์ม
    (ข) "My Listings"  - แสดงเฉพาะรายการที่ตัวเองสร้าง

  ฟังก์ชันเหมือนกับหน้า Bidder:
    - ค้นหาสินค้า
    - กรองตามหมวดหมู่ (15 หมวดหมู่)
    - ดูรายละเอียดสินค้า
    - ตัวนับเวลาถอยหลัง

  เพิ่มเติม:
    - ปุ่มลอย "+" (Floating Action Button) ที่มุมล่างขวา
      สำหรับสร้างรายการประมูลใหม่

--------------------------------------------------------------------------------
8.2 การสร้างรายการประมูล (Create Auction)
--------------------------------------------------------------------------------
  วิธีการ:
    1) กดปุ่ม "+" (Floating Action Button) ที่หน้า Home Seller
       หรือเข้าผ่านเมนู
    2) กรอกข้อมูลทุกช่อง:

       ช่องที่ต้องกรอก:
       +-------------------+----------------------------------------------+
       | ฟิลด์              | รายละเอียด                                    |
       +-------------------+----------------------------------------------+
       | Product Title     | ชื่อสินค้า (จำเป็น)                           |
       | Description       | รายละเอียดสินค้า (จำเป็น)                     |
       | Product Image     | อัปโหลดรูปสินค้า (จำเป็น, มี Preview)         |
       | Starting Price    | ราคาเริ่มต้น (ต้อง > 0)                       |
       | Bid Step          | ขั้นต่ำในการเพิ่มราคา (จำนวนเต็ม, ค่าเริ่มต้น 100) |
       | Category          | หมวดหมู่สินค้า (เลือกจาก 15 หมวดหมู่)        |
       | Start Time        | วันเวลาเริ่มประมูล (ต้องเป็นเวลาในอนาคต)     |
       | End Time          | วันเวลาสิ้นสุดประมูล (ต้องหลังเวลาเริ่ม)     |
       +-------------------+----------------------------------------------+

    3) กดปุ่ม "Create Auction"
    4) เมื่อสร้างสำเร็จ:
       - แสดง "Created successfully!"
       - ระบบนำทางกลับหน้า Seller Home ภายใน 1.8 วินาที

  เงื่อนไข:
    - ทุกช่องเป็นข้อมูลที่จำเป็น (Required)
    - ราคาเริ่มต้นต้องมากกว่า 0
    - Bid Step ต้องเป็นจำนวนเต็ม
    - Start Time ต้องเป็นเวลาในอนาคต
    - End Time ต้องหลัง Start Time
    - รูปภาพรองรับขนาดสูงสุด 10 MB (เก็บแบบ base64)

--------------------------------------------------------------------------------
8.3 การแก้ไขรายการประมูล (Edit Auction)
--------------------------------------------------------------------------------
  เงื่อนไข:
    - สามารถแก้ไขได้เฉพาะ "ก่อน" การประมูลเริ่มต้นเท่านั้น
    - เมื่อประมูลเริ่มแล้วจะไม่สามารถแก้ไขได้

  ฟิลด์ที่แก้ไขได้:
    - Product Title (ชื่อสินค้า)
    - Description (รายละเอียด)

  วิธีการ:
    1) เข้าดู "My Listings" ที่หน้า Seller Home
    2) คลิกที่รายการประมูลที่ต้องการแก้ไข
    3) กดปุ่ม "Edit"
    4) แก้ไขข้อมูลในแบบฟอร์ม
    5) กด "Save" เพื่อบันทึก

--------------------------------------------------------------------------------
8.4 การดูรายการประมูลของตนเอง
--------------------------------------------------------------------------------
  ในหน้า Profile ของผู้ขาย แบ่งเป็น 2 ส่วน:

  (ก) Active Listings - รายการที่กำลังดำเนินอยู่
      - สินค้าที่ยังไม่ถึง End Time
      - แสดงเวลาถอยหลัง
      - แสดงราคาปัจจุบัน

  (ข) History Listings - รายการที่จบแล้ว
      - สินค้าที่ผ่าน End Time แล้ว
      - แสดงราคาสุดท้าย
      - แสดงชื่อผู้ชนะ (ถ้ามี)

--------------------------------------------------------------------------------
8.5 หลังประมูลจบ - ติดต่อผู้ชนะ
--------------------------------------------------------------------------------
  เมื่อการประมูลสิ้นสุดลง:
    - จะมีปุ่ม "💬 Chat with Winner" ปรากฏ
    - กดเพื่อเปิดการสนทนากับผู้ชนะประมูล
    - สามารถตกลงเรื่อง:
      * วิธีการชำระเงิน
      * ที่อยู่จัดส่ง
      * รายละเอียดสินค้าเพิ่มเติม

--------------------------------------------------------------------------------
8.6 การดูโปรไฟล์และสถิติ (Seller Profile)
--------------------------------------------------------------------------------
  แสดงข้อมูล:
    - รูปโปรไฟล์
    - Display Name
    - Email
    - วันที่สมัครสมาชิก
    - คะแนน:
      * จำนวน Like (👍 สีเขียว) จากผู้ซื้อ
      * จำนวน Dislike (👎 สีแดง) จากผู้ซื้อ
    - รายการ Active Listings
    - รายการ History Listings

--------------------------------------------------------------------------------
8.7 การแชท (Chat) - สำหรับผู้ขาย
--------------------------------------------------------------------------------
  ทำงานเหมือนกับ Bidder:
    - ส่งข้อความและรูปภาพ
    - สนทนาเกี่ยวกับรายการประมูลเฉพาะ
    - ใช้ Floating Chat หรือหน้าแชทเต็มจอ

  (ดูรายละเอียดเพิ่มเติมในหัวข้อ 10: ระบบแชท)

--------------------------------------------------------------------------------
8.8 การจัดการโปรไฟล์ (Seller)
--------------------------------------------------------------------------------
  เหมือนกับ Bidder:
    - แก้ไข Display Name
    - แก้ไข Email
    - เปลี่ยน Password
    - อัปโหลดรูปโปรไฟล์ใหม่

9. ฟังก์ชันสำหรับผู้ดูแลระบบ (Admin)
================================================================================

  Admin Dashboard มี 4 แท็บหลัก โดยมี Sidebar เมนูด้านซ้าย:
    - 🏠 Home              (แดชบอร์ดหลัก)
    - 👥 User Management   (จัดการผู้ใช้)
    - ✉️ Reports           (จัดการรายงาน)
    - 📋 Auction Logs      (บันทึกการประมูล)
    - 🚪 Logout            (ออกจากระบบ)

  ส่วนบน (Header) แสดง:
    - โลโก้ JusBid
    - ชื่อ "Admin Dashboard"
    - ไอคอนโปรไฟล์

--------------------------------------------------------------------------------
9.1 แดชบอร์ดหลัก (Admin Home)
--------------------------------------------------------------------------------
  แสดงข้อมูลสถิติภาพรวม:
    - จำนวนรายการประมูลทั้งหมดในระบบ
    - จำนวนผู้ใช้ที่ลงทะเบียนทั้งหมด
    - แสดงรายการประมูลทั้งหมดในรูปแบบ Grid View
    - สรุปข้อมูลสำคัญ

--------------------------------------------------------------------------------
9.2 การจัดการผู้ใช้ (User Management)
--------------------------------------------------------------------------------
  หน้าจอ:
    - ช่องค้นหาผู้ใช้ (ค้นจาก ID, ชื่อ, Username, Email)
    - ตารางแสดงผู้ใช้ทั้งหมด

  คอลัมน์ในตาราง:
    +-------------+------------------------------------------+
    | คอลัมน์      | รายละเอียด                                |
    +-------------+------------------------------------------+
    | User ID     | รหัสผู้ใช้                                |
    | Name        | ชื่อที่แสดง (Display Name)                |
    | Email       | อีเมล                                    |
    | Role        | บทบาท (Buyer/Seller)                     |
    | Status      | สถานะ (Active / Suspended)               |
    | Actions     | ปุ่มดำเนินการ                             |
    +-------------+------------------------------------------+

  ปุ่มดำเนินการต่อผู้ใช้แต่ละคน:

  (ก) View - ดูรายละเอียดผู้ใช้
      เปิด Modal แสดง:
      - รูปโปรไฟล์
      - Display Name, Username, Email
      - บทบาทและวันที่สมัคร
      - สถานะ (Active/Suspended) พร้อม Badge สี
      - ถ้าเป็น Bidder: จำนวนรายการที่ชนะประมูล
      - ถ้าเป็น Seller: จำนวนรายการที่สร้าง, จำนวน Like, จำนวน Dislike

  (ข) Suspend - ระงับบัญชีผู้ใช้
      - กดปุ่ม "Suspend" จะแสดง Dialog ยืนยัน
      - ยืนยัน -> บัญชีจะถูกระงับ
      - ผู้ใช้ที่ถูกระงับจะไม่สามารถเข้าสู่ระบบได้
      - ข้อจำกัด:
        * ไม่สามารถระงับตัวเอง
        * ไม่สามารถระงับบัญชี Admin อื่น

  (ค) Activate - เปิดใช้งานบัญชี
      - ใช้กับบัญชีที่ถูกระงับ (Suspended)
      - กดเพื่อเปิดใช้งานบัญชีอีกครั้ง
      - ผู้ใช้จะสามารถเข้าสู่ระบบได้อีกครั้ง

  (ง) Send Message - ส่งข้อความถึงผู้ใช้
      - Admin สามารถส่งข้อความตรงถึงผู้ใช้คนใดก็ได้
      - ข้อความจะถูกเก็บในฐานข้อมูล

--------------------------------------------------------------------------------
9.3 การจัดการรายงาน (Report Management)
--------------------------------------------------------------------------------
  หน้าจอ:
    - ตัวกรองสถานะ: All / Pending / Resolved / Rejected
    - ช่องค้นหา (ค้นจาก Report ID, ประเภทรายงาน)
    - ตารางแสดงรายงานทั้งหมด

  คอลัมน์ในตาราง:
    +-------------+------------------------------------------+
    | คอลัมน์      | รายละเอียด                                |
    +-------------+------------------------------------------+
    | Report ID   | รหัสรายงาน                               |
    | Type        | ประเภทปัญหา                              |
    | Reporter    | ผู้รายงาน                                |
    | Target      | ผู้ถูกรายงาน                              |
    | Status      | สถานะ (Pending/Resolved/Rejected)        |
    | Date        | วันที่รายงาน                              |
    | Actions     | ปุ่มดำเนินการ                             |
    +-------------+------------------------------------------+

  ประเภทรายงาน:
    - Auction Fraud          (การฉ้อโกง)
    - Non-delivery           (ไม่ส่งสินค้า)
    - Item Mismatch          (สินค้าไม่ตรง)
    - Inappropriate Behavior (พฤติกรรมไม่เหมาะสม)
    - Other                  (อื่นๆ)

  การดำเนินการกับรายงาน:
    เมื่อคลิกที่รายงาน จะเปิดแผงรายละเอียดแสดง:
    - ประเภทรายงาน
    - ข้อมูลผู้รายงาน
    - ข้อมูลผู้ถูกรายงาน
    - วันที่
    - รายละเอียด/คำอธิบาย
    - สถานะปัจจุบัน

    ปุ่มดำเนินการ (แสดงเมื่อสถานะเป็น Pending):
      ✅ Mark as Resolved    - ทำเครื่องหมายว่าแก้ไขแล้ว
      ❌ Reject              - ปฏิเสธรายงาน

    ปุ่มดำเนินการ (แสดงเมื่อสถานะเป็น Resolved/Rejected):
      🔄 Revert to Pending   - เปลี่ยนกลับเป็นรอดำเนินการ

--------------------------------------------------------------------------------
9.4 บันทึกการประมูล (Auction Logs / Bid History)
--------------------------------------------------------------------------------
  หน้าจอ:
    - ช่องค้นหา (ค้นจาก User ID หรือ Auction ID)
    - ตารางแสดงประวัติการเสนอราคาทั้งหมด

  คอลัมน์ในตาราง:
    +-------------+------------------------------------------+
    | คอลัมน์      | รายละเอียด                                |
    +-------------+------------------------------------------+
    | Log ID      | รหัสบันทึก                               |
    | Auction     | ชื่อรายการประมูล                          |
    | User        | ผู้เสนอราคา                               |
    | Bid Amount  | จำนวนเงินที่เสนอ                          |
    | Time        | วันเวลาที่เสนอ (รูปแบบภาษาไทย)           |
    | Status      | สถานะ (Active = ยังประมูลอยู่ / Ended = จบแล้ว) |
    +-------------+------------------------------------------+

  การใช้งาน:
    - ค้นหาด้วย User ID เพื่อดูประวัติการเสนอราคาของผู้ใช้คนนั้น
    - ค้นหาด้วย Auction ID เพื่อดูประวัติทั้งหมดของรายการประมูลนั้น
    - ข้อมูลเรียงตามเวลาล่าสุด

10. ระบบแชท (Chat System)
================================================================================

ระบบแชทมี 2 รูปแบบ:

--------------------------------------------------------------------------------
10.1 แชทแบบเต็มหน้าจอ (Full Chat Page)
--------------------------------------------------------------------------------
  เข้าถึงโดย:
    - กดปุ่ม "Chat" / "💬" ที่การ์ดสินค้า
    - เปิดจาก Floating Chat -> กดปุ่มขยาย

  ฟังก์ชัน:
    - สนทนากับอีกฝ่ายเกี่ยวกับรายการประมูลเฉพาะ
    - ส่งข้อความตัวอักษร
    - ส่งรูปภาพ (เก็บแบบ base64)
    - ดูประวัติข้อความทั้งหมด
    - Auto-scroll ไปข้อความล่าสุดอัตโนมัติ
    - ข้อความอัปเดตทุก 3 วินาที

  การส่งข้อความ:
    - พิมพ์ในช่องข้อความ -> กด Enter หรือปุ่ม Send
    - Shift + Enter = ขึ้นบรรทัดใหม่ (ไม่ส่ง)

  การส่งรูปภาพ:
    - กดไอคอนแนบไฟล์ (📎)
    - เลือกรูปภาพจากเครื่อง
    - รูปจะถูกส่งอัตโนมัติ

--------------------------------------------------------------------------------
10.2 แชทแบบลอย (Floating Chat)
--------------------------------------------------------------------------------
  ตำแหน่ง: ปุ่มลอยอยู่มุมล่างขวาของทุกหน้า (ยกเว้น Admin)

  3 สถานะ:
    (1) ปิด - แสดงเฉพาะปุ่ม "💬"
    (2) รายการสนทนา - แสดง Panel ด้านข้างพร้อมรายชื่อสนทนา
    (3) สนทนา - แสดงหน้าแชทในรายการที่เลือก

  รายการสนทนาแสดง:
    - ชื่ออีกฝ่าย
    - ชื่อรายการประมูล
    - ข้อความล่าสุด (preview)

  ฟังก์ชัน:
    - คลิกปุ่ม "💬" เพื่อเปิด/ปิด
    - เลือกสนทนาจากรายการ
    - ส่งข้อความและรูปภาพ
    - กดปุ่มขยาย เพื่อเปิดเป็นแชทเต็มหน้าจอ
    - กดปุ่มย่อ เพื่อกลับมาเป็น Panel ด้านข้าง
    - รายการสนทนาอัปเดตอัตโนมัติทุก 5 วินาที
    - ข้อความในสนทนาอัปเดตทุก 3 วินาที

11. แถบนำทาง (Navigation Bar)
================================================================================

  องค์ประกอบ:
  +------+--------+------------------+-------------------+
  | โลโก้ | Home   | ช่องค้นหาสินค้า   | ผู้ใช้/Login/Register |
  +------+--------+------------------+-------------------+

  สำหรับผู้ที่ยังไม่ล็อกอิน:
    - ปุ่ม "Login"
    - ปุ่ม "Register"

  สำหรับผู้ที่ล็อกอินแล้ว:
    - ไอคอนผู้ใช้ (คลิกเพื่อเปิดเมนูดร็อปดาวน์)
    - เมนู:
      * Profile  -> ไปหน้าโปรไฟล์ (ตามบทบาท)
      * Logout   -> ออกจากระบบ

  โลโก้/Home:
    - คลิก "JusBid" หรือ "Home" -> ไปหน้าหลักตามบทบาท:
      * Guest   -> Landing Page
      * Bidder  -> /home-bidder
      * Seller  -> /home-seller
      * Admin   -> /admin

  ช่องค้นหา:
    - พิมพ์ชื่อสินค้าเพื่อค้นหาจากรายการประมูลทั้งหมด
    - ค้นหาแบบเรียลไทม์

12. สถานะการประมูล 3 ระยะ
================================================================================

  ระบบประมูลแบ่งออกเป็น 3 ช่วงเวลา:

  +----------+-----------------------------+-------------------------------------+
  | ระยะ      | เงื่อนไข                     | สิ่งที่แสดง/ทำได้                     |
  +----------+-----------------------------+-------------------------------------+
  | ก่อนเริ่ม  | เวลาปัจจุบัน < Start Time   | - แสดงเวลาเริ่มประมูล                |
  | (Before) |                             | - ไม่สามารถประมูลได้                  |
  |          |                             | - ผู้ขายสามารถแก้ไขรายละเอียดได้      |
  +----------+-----------------------------+-------------------------------------+
  | ระหว่าง   | Start Time <= ปัจจุบัน       | - แสดงเวลาถอยหลัง (HH:MM:SS)        |
  | ประมูล    | < End Time                  | - ปุ่ม "Bid" เปิดใช้งาน              |
  | (During) |                             | - ราคาอัปเดตทุก 2 วินาที             |
  |          |                             | - สามารถเสนอราคาได้                  |
  +----------+-----------------------------+-------------------------------------+
  | หลังจบ    | เวลาปัจจุบัน >= End Time    | - แสดงชื่อผู้ชนะ                     |
  | (After)  |                             | - แสดงราคาสุดท้าย                   |
  |          |                             | - ปุ่ม "Chat" กับอีกฝ่ายปรากฏ        |
  |          |                             | - ไม่สามารถเสนอราคาได้อีก            |
  +----------+-----------------------------+-------------------------------------+

13. ความปลอดภัยและการตรวจสอบสิทธิ์
================================================================================

  การยืนยันตัวตน (Authentication):
    - ใช้ JWT (JSON Web Token)
    - Token มีอายุ 1 ชั่วโมง
    - ส่ง Token ผ่าน Header: Authorization: Bearer <token>
    - เมื่อ Token หมดอายุ ต้องล็อกอินใหม่

  การเก็บข้อมูล Session:
    - Token, userId, role เก็บใน Session Storage ของเบราว์เซอร์
    - ปิดแท็บ/หน้าต่างจะหายไป

  การตรวจสอบบัญชี:
    - ทุกครั้งที่ทำ API Request ระบบจะตรวจสอบว่าบัญชีถูกระงับหรือไม่
    - หากถูกระงับจะได้รับ Error 403 พร้อมข้อความแจ้งเตือน
    - ระบบจะ Redirect กลับหน้า Login อัตโนมัติ

  การควบคุมสิทธิ์ตามบทบาท (Role-Based Access Control):
    - เส้นทาง (Route) ที่ต้องล็อกอินจะถูกปกป้องด้วย PrivateRoute
    - API ของ Admin ต้องผ่าน Middleware requireAdmin
    - ผู้ใช้ไม่สามารถเข้าถึงหน้าของบทบาทอื่น
    - Bidder ไม่สามารถสร้างรายการประมูล
    - Seller ไม่สามารถแก้ไขรายการของคนอื่น

  การตรวจสอบข้อมูล (Validation):
    - Email: ต้องอยู่ในรูปแบบที่ถูกต้อง
    - Password: 8 ตัวอักษร + ตัวเล็ก + ตัวใหญ่ + ตัวเลข
    - Username/Email: ห้ามซ้ำ
    - ราคาประมูล: ต้องเป็นจำนวนเต็ม, >=ราคาปัจจุบัน+ขั้นต่ำ
    - เวลาประมูล: ต้องเป็นอนาคต, สิ้นสุดหลังเริ่มต้น

14. หมายเหตุทางเทคนิค
================================================================================

  ฐานข้อมูล (PostgreSQL):
    - ตาราง users          : ข้อมูลผู้ใช้ทุกคน
    - ตาราง auctions       : รายการประมูลทั้งหมด
    - ตาราง bids           : ประวัติการเสนอราคา
    - ตาราง messages       : ข้อความแชททั้งหมด
    - ตาราง user_ratings   : คะแนน Like/Dislike
    - ตาราง reports        : รายงานปัญหา

  การอัปเดตข้อมูลแบบเรียลไทม์:
    - ราคาประมูล: อัปเดตทุก 2 วินาที (Polling)
    - ข้อความแชท: อัปเดตทุก 3 วินาที
    - รายการสนทนา: อัปเดตทุก 5 วินาที

  รูปภาพ:
    - เก็บในรูปแบบ Base64 ในฐานข้อมูล
    - รองรับขนาดสูงสุด 10 MB ต่อรูป

  การนำทาง (Routing):
    - /                    -> Landing Page (Guest)
    - /login               -> หน้าเข้าสู่ระบบ
    - /register            -> หน้าสมัครสมาชิก
    - /home-bidder         -> หน้าหลัก Bidder
    - /home-seller         -> หน้าหลัก Seller
    - /admin               -> หน้า Admin Dashboard
    - /create-auction      -> สร้างรายการประมูล (Seller)
    - /profile-bidder      -> โปรไฟล์ Bidder
    - /profile-seller      -> โปรไฟล์ Seller
    - /profile/:id         -> Public Profile ของผู้ใช้
    - /chat/:auctionId     -> หน้าแชทสำหรับรายการประมูล
